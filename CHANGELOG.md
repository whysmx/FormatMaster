# 更新日志

所有重要变更都将记录在此文件中。

## [1.0.0] - 2025-01-12

### 新增
- 🎉 首次发布
- ✨ 支持Word文档格式还原功能
- 🌐 Web界面操作，简单易用
- 📑 多模板管理功能
- 📊 格式相似度显示
- 📜 转换历史记录
- 🔄 批量转换支持
- 💾 Windows可执行程序（免安装）

### 功能特性
- 支持字体、段落、标题样式还原
- 支持编号和项目符号
- 支持表格样式
- 支持页面设置
- 保留文档原内容
- 支持模板上传、编辑、删除
- 自动计算格式相似度

### 系统要求
- Windows 10 或更高版本（.exe版本）
- 或 Python 3.7+（源码版本）

## 版本说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**：重大架构变更或不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
